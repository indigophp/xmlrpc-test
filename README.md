**XML-RPC test tool.**


## Install

Via Composer

``` bash
$ composer create-project indigophp/xmlrpc-test
```

## Usage

You can use this simple application to test any XML-RPC endpoint.


## Future plans

- Generate JSON schema based on method signature
- Support authentication (eg. basic auth)
